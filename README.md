# Newsletter - Dark/Light Mode

Um projeto de página de captura para Newsletter com interface responsiva e funcionalidade de troca de tema (Dark/Light mode). Desenvolvido para praticar manipulação de estados e estilização moderna.

#Tecnologias
Este projeto foi desenvolvido com as seguintes tecnologias:

* **HTML5** - Estrutura da página.
* **CSS3** - Estilização, Flexbox e variáveis de tema.
* **JavaScript** - Lógica para troca de tema e interação do formulário.

#Funcionalidades
* Alternador de Tema: Botão para alternar entre os modos claro e escuro, melhorando a experiência do usuário.
* Formulário Validado: Campos de entrada prontos para receber nome e e-mail.
* Interface Responsiva: Adaptável a diferentes tamanhos de tela.
